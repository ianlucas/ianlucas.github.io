MathJax Editor

A Mathematics editor on top of MathJax.

Installation Usage Options API Github

Installation

You can install it through npm, or Yarn:

npm install mathjax-editor or yarn add mathjax-editor

Also you can just include MathJaxEditor.js file.

Usage

First, you need to define a textarea.

<textarea id="myEditor"></textarea>

Then, instantiate a new MathJax Editor.

const editor = new MathJaxEditor({
  el: '#myEditor' // or document.getElementById('myEditor')
});

Now you are done!

Options

MathJax Editor accepts an options object as first argument. These are the avaialable properties:

options.el

This option is required. el is the target textarea that will be replaced with MathJax Editor's input. It can be a selector string or a DOM Element.

options.newLine

Enable new line insertion. Default is false

options.scroll

Enable scroll to display element. Default is false, so the display display-x is hidden.

options.value

Set the initial editor value.

API

editor.insert(char)

This insert a character at cursor position. The allowed characters are numbers (0 to 9), and variables (a to z).

editor.insertSymbol(symbol)

This insert a symbol at cursor position. Currently supported symbols: '+', '-', '=', '<', '>', ',', '.', ':', ';', '?', '(', ')', '[', ']', '|', '%'.

editor.insertCommand(command[, blockCount = 0, brackets = false])

This inserts a command into the editor. blockCount is the quantity of blocks ({}) the command requires (e.q. \sqrt requires 1, and \frac requires 2). Tested commands: Greek symbols, \sqrt, \frac, \geq, \leq, \pm.

editor.moveCursorLeft()

This will move the cursor to the left.

editor.moveCursorRight()

This will move the cursor to the right.

editor.erase()

This will erase the character before the cursor.

editor.focus()

Focus the editor.

editor.blur()

Blur the editor.

editor.on(type, listener)

Listen to a editor event. Currently available: focus, and blur.

editor.getValue()

Get the editor's value.
